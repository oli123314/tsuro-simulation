# tsuro-simulation
This repository was created by Oliver Fife on 2/2/2026 for the purpose of giving public acsees to the source files for the numerical methods and visualization tools for the 1st 2026 Ross mathematics Program Application Problem.

## Requirements
- Python 3.12 (other verions will likely work fine)
- Pygame

## Usage
 - main.py | Entry point. Ideal location to visualize n-cards, calculate n-cards, and apply object methods.
 - card.py | Program of central object Card(). Other methods could be made.
 - global_variables.py | Parameters and initialization.
